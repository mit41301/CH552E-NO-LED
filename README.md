# 8ビットUSBマイコン CH552E

## CH552は、MCS51命令セットと互換性のあるE8051コアMCUを強化したものです。命令の79%が1バイトのシングルサイクル命令で、平均命令速度は標準のMCS51より8～15倍速いです。
